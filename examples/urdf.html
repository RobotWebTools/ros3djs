Simple URDF Example

Run the following commands in the terminal then refresh this page. Note that this page must be run on a web server in order to properly server the Collada models. Consider the following basic web server in Python.

  1. roslaunch pr2_description upload_pr2.launch
  2. rosrun robot_state_publisher robot_state_publisher
  3. rosrun joint_state_publisher joint_state_publisher
  4. rosrun tf2_web_republisher tf2_web_republisher
  5. roslaunch rosbridge_server rosbridge_websocket.launch
